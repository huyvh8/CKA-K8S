# Certified Kubernetes Administrator (CKA) Study Guide

Welcome to your CKA study documentation. This repository is structured according to the official CNCF CKA curriculum. Use the links below to navigate to specific domains.

## Curriculum & Navigation

| Weight | Domain | Description |
| :--- | :--- | :--- |
| **25%** | [Cluster Architecture, Installation & Configuration](./01-Cluster-Architecture-Installation-Configuration/README.md) | Control Plane, Worker Nodes, Etcd, Upgrades |
| **15%** | [Workloads & Scheduling](./02-Workloads-Scheduling/README.md) | Pods, Deployments, Scheduling, ConfigMaps |
| **20%** | [Services & Networking](./03-Services-Networking/README.md) | Services, Ingress, Network Policies, DNS |
| **10%** | [Storage](./04-Storage/README.md) | PV, PVC, StorageClasses |
| **30%** | [Troubleshooting](./05-Troubleshooting/README.md) | Cluster failures, Node issues, Application debugging |
| **Bonus** | [Practice Labs](./99-Practice-Labs/README.md) | Integrated scenarios and complex exercises |

## How to use this guide
1. Go through each directory in order (or by weight).
2. Read the **Key Concepts**.
3. Practice the **Commands**.
4. Complete the **Hands-on Steps**.
5. Refer to **Useful Links** for official documentation.
